## 进度控制

### 主要播放控制

<img src="/images/control-panel.png" alt="控制面板" />

播放控制面板的核心区域包含五个主要按钮：

<CardGroup cols={3}>
  <Card title="上一句" icon="skip-back">
    **功能**：跳转到上一条字幕

    **快捷键**：`A`

    **说明**：
    - 智能跳转到上一句字幕的开始位置
    - 如果当前句子播放时间超过2秒，则重新播放当前句子
    - 支持连续按键快速跳转

  </Card>

  <Card title="向后跳转" icon="rewind">
    **功能**：向后跳转

    **快捷键**：`←`

    **说明**：
    - 固定后退10秒（可在设置中自定义）
    - 超过视频开头时自动跳转到开始
    - 连续按键可累积跳转时间

  </Card>

  <Card title="播放/暂停" icon="play">
    **功能**：控制视频播放状态

    **快捷键**：`空格`

    **说明**：
    - 最重要的播放控制按钮
    - 支持点击视频区域控制
    - 自动恢复功能配合使用

  </Card>

  <Card title="向前跳转" icon="fast-forward">
    **功能**：向前跳转

    **快捷键**：`→`

    **说明**：
    - 向前跳转（可在设置中自定义）
    - 超过视频结尾时自动跳转到结束
    - 支持精确的时间定位

  </Card>

  <Card title="下一句" icon="skip-forward">
    **功能**：跳转到下一条字幕

    **快捷键**：`D`

    **说明**：
    - 智能跳转到下一句字幕的开始位置
    - 自动跳过空白时间段
    - 到达结尾时停止播放

  </Card>
</CardGroup>

## 播放速度控制

### 变速播放

<img src="/images/playback-speed.gif" alt="播放速度控制" />

EchoPlayer 提供精确的播放速度控制，特别适合语言学习：

<Tabs>
  <Tab title="快捷操作">
    - **按钮展示**：主按钮实时显示当前速度（保留两位小数，如 `1.25x`）
    - **左键单击**：若已收藏常用速度，则按收藏顺序循环切换；未收藏时展开菜单
    - **即时生效**：选择后立即应用到播放器，并同步更新按钮显示
  </Tab>

  <Tab title="预设速度">
    | 速度 | 场景 |
    |------|------|
    | **0.25x** | 精听拆分语音 |
    | **0.5x** | 慢速理解对话 |
    | **0.75x** | 细致学习 |
    | **1.0x** | 标准播放 |
    | **1.25x** | 略快复习 |
    | **1.5x** | 高效回放 |
    | **1.75x** | 快速浏览 |
    | **2.0x** | 极速扫读 |
  </Tab>

  <Tab title="收藏常用速度">
    - **收藏方式**：在菜单中 `Ctrl+点击`（macOS 为 `Control+点击`）或右键点击速度项可收藏/取消
    - **视觉提示**：收藏项带有黄色描边与圆点标记，当前速度则以主题色高亮
    - **循环切换**：左键单击主按钮即可在收藏列表中循环；收藏为空时保留默认行为
    - **状态保持**：收藏配置存于播放器状态内，多次会话依旧可用
  </Tab>
</Tabs>

> 提示：播放器会将速度限制在 `0.25x ~ 3.0x` 的安全范围内，防止误操作导致异常速度。

### 播放速度的学习建议

<CardGroup cols={2}>
  <Card title="初学者建议" icon="graduation-cap">
    - **0.5x - 0.75x**：初次接触新内容
    - **0.75x - 1.0x**：熟悉内容复习
    - **逐步提升**：随着理解程度提高速度
    - **重点慢放**：难点部分使用慢速
  </Card>

  <Card title="进阶技巧" icon="star">
    - **1.0x - 1.25x**：正常到略快的练习
    - **1.5x - 2.0x**：快速浏览和复习
    - **变速训练**：交替使用不同速度
    - **实时调整**：根据内容难度随时调整
  </Card>
</CardGroup>

## 循环播放功能

### 单句循环

<img src="/images/loop-control.png" alt="循环控制" />

EchoPlayer 的特色功能 - 单句循环播放：

<AccordionGroup>
  <Accordion title="基本操作">
    - **启用循环**：点击循环按钮或按 `R` 键

  <Accordion title="学习应用">

    - **语调模仿**：练习语音语调
    - **词汇记忆**：强化重要词汇印象
    - **语法理解**：分析复杂句子结构
  </Accordion>

### 常见问题解决

<AccordionGroup>
  <Accordion title="播放卡顿问题">
    **可能原因：**
    - 视频文件分辨率过高
    - 硬件解码未启用
    - 系统资源不足

    **解决方案：**
    - 降低播放质量设置
    - 启用硬件加速
    - 关闭其他占用资源的程序
    - 使用较低分辨率的视频文件

  </Accordion>

  <Accordion title="音画不同步">
    **可能原因：**
    - 视频文件本身问题
    - 解码器兼容性问题
    - 音频设备延迟

    **解决方案：**
    - 使用音频同步调整功能
    - 更换视频文件
    - 检查音频设备设置
    - 重启播放器

  </Accordion>

  <Accordion title="快捷键不响应">
    **可能原因：**
    - 焦点不在播放器窗口
    - 输入法冲突
    - 其他软件占用快捷键

    **解决方案：**
    - 点击播放器窗口获取焦点
    - 切换到英文输入法
    - 检查快捷键冲突
    - 重新自定义快捷键

  </Accordion>
</AccordionGroup>

EchoPlayer 的字幕系统是其核心优势，专为语言学习优化，提供丰富的字幕管理和显示功能。

## 字幕文件支持

### 支持的格式

<CardGroup cols={2}>
  <Card title="SRT (.srt)" icon="file-text">
    **最常用的字幕格式**
    - 简洁的文本格式
    - 广泛的兼容性
    - 时间轴精确
    - 适合大多数场景
  </Card>

<Card title="VTT (.vtt)" icon="file-video">
  **Web 标准字幕格式** - HTML5 视频标准 - 支持样式和定位 - Web 平台友好 - 现代化设计
</Card>

<Card title="ASS/SSA (.ass/.ssa)" icon="file-code">
  **高级字幕格式** - 丰富的样式支持 - 动画和特效 - 精确的定位控制 - 专业制作标准
</Card>

  <Card title="JSON (.json)" icon="file-lines">
    **EchoPlayer 专用格式**
    - 结构化数据存储
    - 扩展元数据支持
    - 学习进度记录
    - 自定义标记
  </Card>
</CardGroup>

### 文件命名规则

EchoPlayer 采用智能的字幕文件检测机制：

```text
视频文件：movie.mp4

自动检测的字幕文件：
├── movie.srt                 # 默认字幕
├── movie.zh.srt             # 中文字幕
├── movie.en.srt             # 英文字幕
├── movie.zh-CN.srt          # 简体中文
├── movie.zh-TW.srt          # 繁体中文
├── movie.ja.srt             # 日文字幕
├── movie.ko.srt             # 韩文字幕
└── movie.forced.srt         # 强制字幕
```

<Info>系统会按照优先级自动选择字幕文件，您也可以手动切换或添加字幕。</Info>

## 字幕导入和管理

### 自动导入

启动视频播放时，EchoPlayer 会自动扫描：

<Steps>
  <Step title="同目录扫描">扫描视频文件所在目录的同名字幕文件</Step>
  <Step title="子目录查找">检查 `subtitles/`、`subs/`、`字幕/` 等常见字幕目录</Step>
  <Step title="语言识别">根据文件名识别字幕语言和类型</Step>
  <Step title="自动加载">按优先级自动加载最佳字幕</Step>
</Steps>

### 手动导入

如果需要手动添加字幕：

<Tabs>
  <Tab title="播放时导入">
    1. 在播放界面点击字幕按钮 <Icon icon="captions" size={16} />
    2. 选择"导入字幕文件"
    3. 浏览并选择字幕文件
    4. 确认加载并选择语言类型
  </Tab>

<Tab title="拖拽导入">
  1. 直接拖拽字幕文件到播放器窗口 2. 系统自动识别并加载 3. 可同时拖拽多个字幕文件 4. 支持批量处理
</Tab>

  <Tab title="在线下载">
    1. 在字幕菜单中选择"在线搜索"
    2. 自动匹配视频信息
    3. 从字幕库中下载合适字幕
    4. 一键导入到播放器
  </Tab>
</Tabs>

<img src="/images/subtitle-list-panel.png" alt="字幕列表面板" />

#### 核心功能

<AccordionGroup>
  <Accordion title="字幕浏览">
    - **完整列表显示**：展示所有字幕条目
    - **当前高亮**：正在播放的字幕高亮显示
    - **自动滚动**：跟随播放进度自动滚动

  <Accordion title="搜索和过滤">
    - **文本搜索**：快速查找包含特定词汇的字幕
    - **时间范围过滤**：按时间段筛选字幕
    - **语言过滤**：在多语言字幕中筛选
    - **高亮显示**：搜索结果高亮标记
  </Accordion>
</AccordionGroup>

### 显示模式

根据字幕文件和用户设置，面板支持多种显示模式：

<Tabs>
  <Tab title="单语言模式">
    ```
    [00:01:23] Hello, how are you today?
    [00:01:26] I'm fine, thank you.
    [00:01:30] What about you?
    [00:01:33] I'm doing great!
    ```

    **特点：**
    - 简洁清晰的单语言显示
    - 适合单一字幕文件
    - 节省屏幕空间

  </Tab>

  <Tab title="双语言模式">
    ```
    [00:01:23] Hello, how are you today?
            你好，你今天怎么样？

    [00:01:26] I'm fine, thank you.
            我很好，谢谢你。

    [00:01:30] What about you?
            你呢？
    ```

    **特点：**
    - 原文和译文对照显示
    - 学习效果最佳
    - 便于理解和记忆

  </Tab>

  <Tab title="紧凑模式">
    ```
    [01:23] Hello, how are you? | 你好吗？
    [01:26] I'm fine, thanks. | 我很好，谢谢。
    [01:30] What about you? | 你呢？
    ```

    **特点：**
    - 减少垂直空间占用
    - 适合小屏幕设备
    - 提高信息密度

  </Tab>
</Tabs>

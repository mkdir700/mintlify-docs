EchoPlayer 提供丰富的键盘快捷键，帮助您更高效地进行语言学习。所有快捷键都可以根据个人习惯进行自定义。

## 基础播放控制

<CardGroup cols={2}>
  <Card title="播放/暂停" icon="play">
    **快捷键**: `空格键 (Space)`

    **功能**: 切换视频播放状态

    **说明**: 最常用的控制键，随时可以暂停或继续播放
  </Card>

  <Card title="向前跳转" icon="skip-forward">
    **快捷键**: `→ (右箭头)`

    **功能**: 向前跳转（通常是下一句字幕）

    **说明**: 精确跳转到下一句字幕的开始位置
  </Card>

  <Card title="向后跳转" icon="skip-back">
    **快捷键**: `← (左箭头)`

    **功能**: 向后跳转（通常是上一句字幕）

    **说明**: 精确跳转到上一句字幕的开始位置
  </Card>

  <Card title="全屏切换" icon="maximize">
    **快捷键**: `F`

    **功能**: 切换全屏模式

    **说明**: 进入沉浸式学习环境
  </Card>
</CardGroup>

## 自定义快捷键

### 如何自定义快捷键

<Steps>
  <Step title="打开设置">
    点击设置按钮进入应用设置界面
  </Step>
  <Step title="选择快捷键设置">
    在设置界面中找到"快捷键设置"选项
  </Step>
  <Step title="修改快捷键">
    点击想要修改的功能，然后按下新的按键组合
  </Step>
  <Step title="保存设置">
    确认修改后，设置会自动保存并立即生效
  </Step>
</Steps>

### 快捷键设置规则

<Warning>
**注意事项**:
- 避免与系统快捷键冲突
- 建议使用容易记忆的按键组合
- 某些系统级快捷键（如 `Escape`）无法修改
- 快捷键修改后立即生效，无需重启应用
</Warning>

<Tip>
**推荐配置**:
- 将最常用的功能设置为单键快捷键
- 为相关功能使用相邻的按键（如 `A`/`D` 用于前后导航）
- 保持组合键的一致性（如都使用 `Shift+` 前缀）
</Tip>

## 使用技巧

### 高效学习组合

<CardGroup cols={2}>
  <Card title="精听练习" icon="headphones">
    **推荐组合**: `S` + `R` + `↓`

    1. 使用 `S` 重播当前句子
    2. 按 `R` 开启单句循环
    3. 按 `↓` 降低音量到舒适水平
    4. 专注听懂每个单词
  </Card>

  <Card title="快速浏览" icon="zap">
    **推荐组合**: `Shift+]` + `D` + `→`

    1. 用 `Shift+]` 提高播放速度
    2. 用 `D` 快速跳转字幕
    3. 用 `→` 按句子浏览内容
    4. 找到需要精听的部分
  </Card>

  <Card title="笔记记录" icon="edit">
    **推荐组合**: `Space` + `Ctrl+C` + `A`

    1. 按 `Space` 暂停播放
    2. 用 `Ctrl+C` 复制当前字幕
    3. 记录到笔记应用中
    4. 按 `A` 返回上一句继续学习
  </Card>

  <Card title="沉浸学习" icon="focus">
    **推荐组合**: `F` + `R` + `S`

    1. 按 `F` 进入全屏模式
    2. 用 `R` 开启循环播放
    3. 用 `S` 重复难懂的句子
    4. 创造沉浸式学习环境
  </Card>
</CardGroup>

### 常见问题解决

<AccordionGroup>
  <Accordion title="快捷键不响应">
    **可能原因**:
    - 应用窗口失去焦点
    - 输入法冲突
    - 与其他软件快捷键冲突

    **解决方案**:
    - 点击播放器窗口获取焦点
    - 切换到英文输入模式
    - 检查并关闭冲突的软件
    - 自定义快捷键避免冲突
  </Accordion>

  <Accordion title="快捷键设置无法保存">
    **可能原因**:
    - 文件权限问题
    - 存储空间不足
    - 设置文件损坏

    **解决方案**:
    - 以管理员权限运行应用
    - 清理磁盘空间
    - 重置快捷键设置到默认值
  </Accordion>

  <Accordion title="某些快捷键无法自定义">
    **说明**:
    - 系统级快捷键（如 `Escape`）无法修改
    - 这些快捷键标记为"系统"类型
    - 确保核心功能的稳定性

    **建议**:
    - 了解哪些快捷键无法修改
    - 围绕固定快捷键设计个人习惯
  </Accordion>
</AccordionGroup>

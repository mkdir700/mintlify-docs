欢迎使用 EchoPlayer！本指南将帮助您快速掌握基本操作，开始高效的视频语言学习。

## 前提条件

在开始之前，请确保：

<Steps>
  <Step title="已安装 EchoPlayer">
    如果还未安装，请查看 [安装指南](/installation)
  </Step>
  <Step title="准备学习材料">
    准备一个视频文件和对应的字幕文件（可选）
  </Step>
</Steps>

## 第一步：启动并熟悉界面

### 启动应用

根据您的操作系统启动 EchoPlayer：

<Tabs>
  <Tab title="Windows">
    - 从开始菜单搜索 "EchoPlayer"
    - 或双击桌面上的 EchoPlayer 图标
  </Tab>
  <Tab title="macOS">
    - 在应用程序文件夹中找到 EchoPlayer
    - 或使用 Spotlight 搜索 "EchoPlayer"
  </Tab>
  <Tab title="Linux">
    - 在应用程序菜单中找到 EchoPlayer
    - 或在终端中运行 `echoplayer`
  </Tab>
</Tabs>

### 界面概览

启动后您将看到 EchoPlayer 的主界面：

<img src="/images/main-interface.png" alt="EchoPlayer 主界面" />

主要区域包括：
- **侧边栏** - 视频库和导航
- **主内容区** - 视频列表和播放区域
- **顶部菜单** - 设置和功能按钮

## 第二步：添加视频

### 方法一：直接添加文件

<Steps>
  <Step title="点击添加按钮">
    在主界面点击 "添加视频" 按钮
  </Step>
  <Step title="选择视频文件">
    在文件对话框中选择您要学习的视频文件

    <Info>
    支持的格式：MP4、AVI、MKV、MOV、WMV、FLV 等
    </Info>
  </Step>
  <Step title="确认添加">
    点击"打开"将视频添加到媒体库
  </Step>
</Steps>

### 方法二：拖拽添加

您也可以直接将视频文件拖拽到 EchoPlayer 窗口中：

1. 打开文件管理器
2. 找到目标视频文件
3. 拖拽文件到 EchoPlayer 主界面
4. 松开鼠标完成添加

<Tip>
可以同时拖拽多个视频文件进行批量添加！
</Tip>

## 第三步：添加字幕

字幕是语言学习的重要工具。EchoPlayer 支持多种方式添加字幕：

### 自动检测

EchoPlayer 会自动查找与视频同名的字幕文件：

```
电影.mp4          # 视频文件
电影.srt          # 自动检测的字幕
电影.zh.srt       # 中文字幕
电影.en.srt       # 英文字幕
```

### 手动添加

如果需要手动添加字幕：

<Steps>
  <Step title="进入播放界面">
    点击视频开始播放
  </Step>
  <Step title="点击字幕按钮">
    在播放控制栏找到字幕图标 <Icon icon="captions" size={16} />
  </Step>
  <Step title="选择字幕文件">
    点击"导入字幕"，选择对应的字幕文件
  </Step>
  <Step title="确认加载">
    字幕将立即加载并显示
  </Step>
</Steps>

### 支持的字幕格式

<CardGroup cols={2}>
  <Card title="SRT" icon="file-text">
    最常见的字幕格式
  </Card>
  <Card title="VTT" icon="file-video">
    Web 标准字幕格式
  </Card>
  <Card title="ASS/SSA" icon="file-code">
    支持样式的字幕格式
  </Card>
  <Card title="JSON" icon="file-lines">
    EchoPlayer 专用格式
  </Card>
</CardGroup>

## 第四步：开始语言学习

现在您可以开始使用 EchoPlayer 的特色功能进行语言学习了！

### 基本播放控制

<img src="/images/player-controls.png" alt="播放控制栏" />

主要控制按钮：

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| <Icon icon="skip-back" size={16} /> | 上一句字幕 | `A` |
| <Icon icon="rewind" size={16} /> | 向后跳转 | `←` |
| <Icon icon="play" size={16} />/<Icon icon="pause" size={16} /> | 播放/暂停 | `空格` |
| <Icon icon="fast-forward" size={16} /> | 向前跳转 | `→` |
| <Icon icon="skip-forward" size={16} /> | 下一句字幕 | `D` |

### 逐句学习模式

EchoPlayer 的核心功能是逐句精听：

<Steps>
  <Step title="启用自动暂停">
    在设置中开启"句子结束后自动暂停"功能
  </Step>
  <Step title="设置暂停延迟">
    根据需要调整暂停延迟时间（推荐 1-3 秒）
  </Step>
  <Step title="开始学习">
    播放视频，系统会在每句字幕结束后自动暂停
  </Step>
  <Step title="消化理解">
    利用暂停时间理解刚才听到的内容
  </Step>
</Steps>

### 单句循环

对于难理解的句子，可以使用单句循环功能：

1. 播放到目标句子
2. 点击循环按钮 <Icon icon="repeat" size={16} />
3. 该句子将重复播放直到您取消循环

### 字幕显示模式

根据学习需要选择合适的字幕显示模式：

<Tabs>
  <Tab title="隐藏字幕">
    专注于听力训练，不显示任何字幕
  </Tab>
  <Tab title="原文字幕">
    只显示原语言字幕，适合进阶学习者
  </Tab>
  <Tab title="译文字幕">
    只显示翻译字幕，帮助理解内容
  </Tab>
  <Tab title="双语字幕">
    同时显示原文和译文，适合初学者
  </Tab>
</Tabs>

## 第五步：个性化设置

根据您的学习习惯调整播放器设置：

### 播放设置

<AccordionGroup>
  <Accordion title="自动暂停设置">
    - **启用自动暂停**：每句字幕结束后自动暂停
    - **暂停延迟时间**：控制暂停前的延迟（0-5 秒）
    - **自动恢复播放**：设置自动恢复播放的时间间隔
  </Accordion>

  <Accordion title="字幕设置">
    - **字体大小**：调整字幕文字大小
    - **显示位置**：选择字幕在画面中的位置
    - **背景样式**：设置字幕背景（透明/模糊/纯色）
    - **显示模式**：选择字幕语言显示方式
  </Accordion>

  <Accordion title="快捷键设置">
    自定义各种操作的键盘快捷键，提高学习效率
  </Accordion>
</AccordionGroup>

## 常用快捷键

掌握这些快捷键可以大大提高学习效率：

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 播放/暂停 | `空格` | 控制视频播放状态 |
| 向后跳转 | `←` | 向后跳转（通常是上一句字幕） |
| 向前跳转 | `→` | 向前跳转（通常是下一句字幕） |
| 上一句字幕 | `A` | 跳转到上一条字幕 |
| 下一句字幕 | `D` | 跳转到下一条字幕 |
| 单句循环 | `R` | 开启/关闭当前句子循环 |
| 重播当前字幕 | `S` | 重新播放当前字幕 |
| 全屏切换 | `F` | 进入/退出全屏模式 |
| 音量增加 | `↑` | 增加音量 |
| 音量减少 | `↓` | 减少音量 |
